## 🎯 测试步骤
### 第一步：安装APP
- 找到APK文件并安装到Android手机
- 需要开启"未知来源应用"安装
### 第二步：基础测试
1. **打开APP** - 看到主界面
2. **点击右下角浮动按钮** - 模拟屏幕开关
3. **观察状态变化** - 颜色和文字会改变
### 第三步：自动管理测试
1. **开启自动管理开关**
2. **点击右下角按钮关闭屏幕**
3. **观察** - 系统自动限制3个应用
4. **再次点击按钮开启屏幕**
5. **观察** - 系统自动恢复权限
### 第四步：手动控制测试
1. **点击"恢复权限"按钮**
2. **观察** - 受管应用数量变为0
3. **点击"停止应用"按钮**
4. **观察** - 受管应用数量变为5
### 第五步：观察统计和日志
- 查看顶部的实时统计
- 查看操作日志记录
## 📊 功能说明
### 屏幕状态监听
- 🟢 绿色背景：屏幕开启
- 🟠 橙色背景：屏幕关闭
- 自动根据状态调整显示
### 智能权限管理
- 屏幕关闭时：限制3个后台应用
- 屏幕开启时：恢复所有权限
- 实时显示管理状态
### 手动控制
- **恢复权限**：一键恢复所有应用
- **停止应用**：一键停止非必要应用
- 即时反馈，操作记录
### 实时统计
- 受管应用数量
- 屏幕状态
- 自动模式状态
### 操作日志
- 系统启动记录
- 屏幕状态变化
- 自动管理状态
- 当前系统状态
